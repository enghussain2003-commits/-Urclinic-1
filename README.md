# نظام إدارة العيادات

تطبيق ويب لإدارة العيادات الطبية، مبني بـ React و Vite.

## المميزات

- تسجيل دخول وتسجيل حسابات جديدة
- إدارة ملفات المرضى
- عرض قائمة المرضى
- جدولة المواعيد
- صفحات خاصة بالكادر الطبي

## تقنيات المشروع

- **React** — بناء واجهة المستخدم
- **Vite** — بيئة التطوير والبناء
- **Supabase** — قاعدة البيانات والمصادقة
- **ESLint** — جودة الكود

## تشغيل المشروع

```bash
npm install
npm run dev
```
